Roteiro de aula

Como explicar o SaaS

  1. Abra este sistema em dois navegadores ou dois dispositivos.
  2. Publique um aviso no painel do professor.
  3. Mostre que o painel do aluno recebe o mesmo dado centralizado.
  4. Explique que o usuário consome o software pronto, sem instalar nada.
  5. Mostre a diferença entre o modo local e o modo em nuvem.
